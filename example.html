De-Caesar

JavaScript Ceasear Shift Decoder by Caleb O'Leary

A Caesar cipher is a method of enciphering messages by shifting each letter x letters down in the alphabet. With a shift of 1, 'a' becomes 'b', 'caleb' becomes 'dbmfc', etc. This tool was originally developed to show all possible Caesar shifts of a given string (I needed to solve a puzzle, that's not cheating is it?). I expanded it to analyze the letter frequency of each result, and pick the most likely answer.

De-Vigenere

JavaScript Vigenere Cipher Decoder by Caleb O'Leary

A Vigenere Cipher is a method of enciphering messages by using any amount of interwoven Caesar ciphers. Rather than encoding a message with a number, the message is encoded with a 'key string'. For a key string of 'caleb', the first letter of the message would be shifted 3 positions (since 'c' is the third letter of the alphabet), the second letter 1 space (for 'a') etc. The sixth letter would loop back to the front of the key, shifting by 3 spaces again for 'c'. The Vigenere Cipher was considered unbreakable for several centuries before being broken in the mid-1800's by Charles Babbage, who found that you could exploit the repetitive nature of the key, which would eventually produce repeats in the text. To crack a Vigenere ciphered message, this program looks for repetitions in the ciphertext, and counts the letters between them. This number and its factors are then considered potential key lengths. For a key length of 3, we would run a frequency analysis on every 3rd letter (the same function that cracks Caesar ciphers) and return the most likely shift for that group. The program then arranges its top 10 results by the number of real words it recognizes (from a list of the 1000 most commom English words). The cracking functions work best with a long message.
